Лабораторная работа №8

Выполните задание в соответствии с вашим вариантом
№Условия заданий для выполнения лабораторной работы
1Создайте ассоциативный массив с использованием JSON. Отсортируйте массив по убыванию значений ключа. 
2Создайте ассоциативный массив с использованием JSON. Отсортируйте массив по возрастанию значений ключа. 
3Создайте ассоциативный массив с использованием JSON. Выведите запись с наибольшим значением ключа. Удалите эту запись из массива и выведите его на страницу. 
4Создайте ассоциативный массив с использованием JSON. Отсортируйте первую половину записей по убыванию значений ключа, а вторую половину - по возрастанию. 
5Создайте ассоциативный массив с использованием JSON. Напишите функции нахождения максимального значения ключа, минимального значения ключа. Поменяйте местами записи с максимальным и минимальным значением ключа. 
6Создайте ассоциативный массив с использованием JSON. Напишите функцию нахождения ключей с одинаковым значением, привязанных к ним данных. 
7Создайте ассоциативный массив с использованием JSON. Отсортируйте первую половину записей по возрастанию значений ключа, а вторую половину - по убыванию. 
8Создайте ассоциативный массив с использованием JSON. Выведите запись с наибольшим значением ключа. Удалите эту запись из массива и выведите его на страницу. 
9Создайте ассоциативный массив с использованием JSON. Отсортируйте массив по возрастанию значений ключа. 
10Создайте ассоциативный массив с использованием JSON. Отсортируйте массив по убыванию значений ключа.
